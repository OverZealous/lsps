# @qualified/codemirror-workspace Web Worker demo

Demo with [minimal JSON Language Server][json-ls] in Web Worker.
Everything runs in browser.

---

Install `pnpm` if you don't have it installed:

```
npm i -g pnpm
```

In project root:

```
pnpm install
```

In this directory:

```
pnpm start
```
